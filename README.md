# PEAK-CAN-TRC-file-compare
GUI that opens two .trc files and compares them line by line or frame by frame to verify if the payload parts are identical or not


#### 1) Open the executable `TrcCmp_V1_0` in `dist` folder 
![alt text](https://github.com/RamiMarzougui/PEAK-CAN-TRC-file-compare/blob/main/screenshots/trc_cmp_gui_logo.PNG?raw=true)

#### 2) Click on `Open File 1` and select the first .trc file
![alt text](https://github.com/RamiMarzougui/PEAK-CAN-TRC-file-compare/blob/main/screenshots/trc_cmp_gui.PNG?raw=true)

#### 3) Click on `Open File 2` and select the second .trc file
![alt text](https://github.com/RamiMarzougui/PEAK-CAN-TRC-file-compare/blob/main/screenshots/trc_cmp_gui_trc1.PNG?raw=true)

#### 4) Once both .trc files are selected and loaded (number of CAN frames is displayed), click on `Compare`
![alt text](https://github.com/RamiMarzougui/PEAK-CAN-TRC-file-compare/blob/main/screenshots/trc_cmp_gui_trc2.PNG?raw=true)

#### 5) A .txt file contaning all the frame differences between the two files is generated and the number of errors is displayed
![alt text](https://github.com/RamiMarzougui/PEAK-CAN-TRC-file-compare/blob/main/screenshots/trc_cmp_gui_cmpare.PNG?raw=true)

#### 6) The generated differences file can be saved by clicking on `Save` and selecting a location
![alt text](https://github.com/RamiMarzougui/PEAK-CAN-TRC-file-compare/blob/main/screenshots/trc_cmp_gui_result.PNG?raw=true)
